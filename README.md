# 握力測定システム

このアプリは、学生の皆さんが握力測定の結果をその場で入力し、即座に評価やグラフを確認するためのツールです。

## 使い方
1. ブラウザでアプリを開きます。
2. 右手・左手の測定値を入力します。
3. 「判定」ボタンを押すと、平均値や偏差値、評価が表示されます。
4. 「グループに追加」を押すと、クラスやチーム全体のデータを蓄積し、比較グラフを表示できます。

## 注意事項
- ブラウザを閉じたりリロードしたりすると、入力したデータは消去されます。
- データの保存が必要な場合は、画面下の「CSVダウンロード」機能を利用してください。

# 背筋力測定システム

背筋力の測定結果を管理・分析するためのアプリです。

## 使い方
1. 性別を選択し、測定値を入力します。
2. 年齢に応じた評価や偏差値が自動計算されます。
3. チームメイトのデータを次々に追加して、リアルタイムで順位や分布を確認できます。

## 機能
- 個別評価（偏差値・5段階評価）
- グループ統計グラフ
- 測定データの一括ダウンロード（CSV形式）
